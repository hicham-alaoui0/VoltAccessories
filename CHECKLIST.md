# Theme QA Checklist

## 1. Design & Responsiveness
- [ ] Check Mobile Layout (375px+). Ensure horizontal scrolling works for Testimonials.
- [ ] Check Tablet Layout. Ensure grid columns adjust correctly (2 -> 4).
- [ ] Check Desktop Layout. Verify page width and spacing (standard 1200px).
- [ ] Verify Sticky Header behavior on scroll.

## 2. Functionality
- [ ] **Add to Cart**: Test on Desktop and Mobile. Verify "Sticky Add to Cart" works on mobile product page.
- [ ] **Cart Drawer**: Ensure cart opens as a drawer/slide-out.
- [ ] **Navigation**: Test Mega Menu (if configured) and Mobile Menu.
- [ ] **Search**: Verify predictive search works.
- [ ] **Filters**: Test collection filtering tags/price.

## 3. Sections & Blocks
- [ ] **Trust Bar**: Icons render correctly? Text aligned?
- [ ] **Category Tiles**: Hover zoom effect works? Links correct?
- [ ] **Testimonials**: Slider snaps correctly?
- [ ] **FAQ**: Accordions open/close smoothly?

## 4. Performance & SEO
- [ ] **Images**: Ensure all images are lazy-loaded (handled by default in OS 2.0 sections).
- [ ] **H1 Tags**: Ensure only one H1 per page (Template defaults are set).
- [ ] **Speed**: Run Lighthouse/PageSpeed Insights. Aim for >90 Mobile.

## 5. Dropshipping Specifics
- [ ] **Shipping Page**: Verify content is clear.
- [ ] **Tracking Page**: Verify placeholder widget is visible.
- [ ] **Policies**: Ensure Refund/Privacy policies are linked in Footer.
