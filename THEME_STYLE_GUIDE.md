# VoltAccessories Theme Style Guide

## Brand Identity
- **Name**: VoltAccessories
- **Tone**: Premium, Minimal, Tech-focused
- **Core Promise**: High-quality tech accessories with reliable shipping.

## Color System

### Primary Palette
| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Charcoal** | `#1A1A1A` | Primary text, Dark mode background, Primary buttons |
| **Off-White** | `#F5F5F7` | Light mode background, Dark mode text |
| **Electric Blue** | `#2E5CFF` | Accents, Call-to-action, Highlights |

### Color Schemes (Theme Settings)
- **Scheme 1 (Light)**: Off-White Bg / Charcoal Text. Used for standard pages.
- **Scheme 2 (Dark)**: Charcoal Bg / Off-White Text. Used for footer, hero, or dark mode toggle.
- **Scheme 3 (Accent)**: Electric Blue Bg / White Text. Used for banners, sale tags.

## Typography
- **Font Family**: System Sans-Serif (Inter / San Francisco fallback).
- **Scale**:
    - **Headings**: Bold, tight letter spacing.
    - **Body**: Regular, legible, 16px base.
    - **Labels**: Uppercase, tracked out (letter-spacing: 1px), 12px.

## UI Components

### Buttons
- **Primary**: Solid Charcoal (Light mode) or Off-White (Dark mode). Square corners (0px radius).
- **Secondary**: Outline / Ghost style.
- **Accent**: Electric Blue solid.

### Cards (Product/Collection)
- **Style**: Standard (No border).
- **Image**: Clean, no padding.
- **Content**: Left-aligned below image.

### Forms
- **Inputs**: Minimal border (1px), slight opacity (55%), no shadow. 0px radius.
- **Focus State**: Electric Blue border.

## Spacing & Layout
- **Page Width**: 1200px (Settings default).
- **Grid Spacing**: 8px (Tight, modern feel).
- **Section Spacing**: 0px between sections (Seamless flow) or consistent 64px.
